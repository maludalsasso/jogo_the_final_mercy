# jogo_the_final_mercy
Um jogo interativo em texto ambientado em Desterro (Florianópolis) no ano de 1823. Após um acidente misterioso, o jogador acorda em uma antiga casa colonial, onde eventos estranhos começam a acontecer. Guiado por Alzira, uma mulher enigmática, você deverá investigar o ambiente, tomar decisões e descobrir segredos enterrados.
